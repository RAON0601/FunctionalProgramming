# 평가
<p>
평가란 코드가 계산되어 값을 만드는 것을 의미한다.

```js
const a = 1;
const b = 2;
console.log(a + b); // a + b 라는 식이 평가되어 3이라는 값을 만든다.
```

</p>

# 일급함수

일급함수란 값으로 다룰 수 있는 함수를 말한다.

```js
const add5 = a => a + 5;
const add10 = a => a + 10;

console.log(add5(add10(100))); // 115
```

함수를 값으로 다룰 수 있다는 특징을 이용하면 함수에 인자로 함수를 전달할 수 있다.
